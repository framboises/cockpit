Manuel de la tablette Field

Tout ce qu un operateur doit savoir pour utiliser la tablette pendant un evenement. Ce guide est aussi accessible depuis le bouton ? de la topbar de l application.

01Bienvenue

Field est l application installee sur votre tablette. Elle est le compagnon numerique du PC Organisation pendant un evenement (festival, course, manifestation sportive). Elle vous permet de recevoir vos missions, de remonter de l information au PC Org, et d etre localise en permanence.

Ce que la tablette fait pour vous

  • Affiche la carte de l evenement avec votre position en temps reel
  • Vous notifie d une mission a accomplir (sonnerie + vibration)
  • Trace votre passage et vos actions automatiquement (pas besoin de noter)
  • Vous permet d envoyer des photos, des codes scannes, des comptes-rendus
  • Permet de declencher une alerte SOS qui touche immediatement le PC Org et vos collegues patrouilleurs

Ce que la tablette NE fait PAS

La tablette ne remplace pas la radio. Pour toute communication vocale (demande de renfort, transmission urgente, coordination immediate), utilisez la radio. La tablette est un complement ecrit et trace, pas un substitut a la voix.
  • Elle ne gere pas les communications vocales
  • Elle ne se connecte pas automatiquement a Internet : si vous etes hors couverture, certaines actions sont mises en file d attente
  • Elle ne contient pas de donnees confidentielles : si elle est perdue, le PC Org la revoque a distance

02Avant de commencer

Une tablette = une patrouille

Chaque tablette porte un nom fixe attribue par le PC Org au moment de l appairage : T01, T02, Moto-1, etc. Ce nom ne change pas pendant l evenement. Il identifie la patrouille, pas la personne qui tient la tablette.

  +---------+        +---------+        +---------+
  |  T01    |        |  T02    |        |  T03    |
  | tablette|        | tablette|        | tablette|
  +----+----+        +----+----+        +----+----+
       |                  |                  |
       v                  v                  v
  Patrouille 1      Patrouille 2      Patrouille 3
  (l equipage       (l equipage       (l equipage
   peut changer)     peut changer)     peut changer)

Pas de connexion personnelle

Pas de login, pas de mot de passe, pas de badge a scanner. Vous n avez pas a vous identifier sur la tablette. Si vous prenez le relais d un collegue, contentez-vous de saisir l information sur le cahier de releve papier de la patrouille.

Le cahier de releve (organisation humaine)

Comme la tablette ne sait pas qui la tient, le PC Org tient un cahier (papier ou tableur) de qui est sur quelle patrouille a chaque creneau. Quand vous prenez le poste :

  1. Notez votre nom, l heure de prise de poste, l etat de la batterie
  2. Verifiez que la tablette est sur le bon statut (probablement Disponible)
  3. Lisez les dernieres consignes dans la rubrique Messages
  4. A la fin de votre vacation : passez la tablette en Pause et notez l heure de fin

Verifications avant de partir

  • Batterie : la pastille doit etre verte (au-dessus de 30%). Sinon, branchez la tablette sur le powerbank de la patrouille.
  • Reseau : la pastille reseau doit etre verte. Si vous etes en zone blanche, c est normal de la voir rouge ponctuellement.
  • GPS : la pastille GPS passe en vert apres 5 a 30 secondes en exterieur. Si elle reste rouge, sortez du batiment et patientez.

03Demarrage rapide (5 minutes)

Vous venez de recevoir une tablette neuve, jamais appairee. Suivez ces etapes dans l ordre.

Etape 1 - Recevoir le code

Le PC Organisation vous donne un code a 6 chiffres (par radio, par SMS ou de la main a la main). Ce code n est valide que 15 minutes. Si vous l avez recu il y a plus longtemps, demandez-en un nouveau.

Etape 2 - Allumer la tablette et ouvrir Field

  1. Allumez la tablette (bouton power sur le cote)
  2. Sur l ecran d accueil, tapez sur l icone Field (pictogramme bleu)
  3. Si l icone n est pas la, ouvrez le navigateur et tapez cockpit.lemans.org/field

Etape 3 - Saisir le code

L ecran d appairage affiche un pave numerique :

  +----------------------------------+
  |        COCKPIT FIELD             |
  |  Appairage de la tablette        |
  |                                  |
  |   [_] [_] [_] [_] [_] [_]        |
  |                                  |
  |    +---+ +---+ +---+             |
  |    | 1 | | 2 | | 3 |             |
  |    +---+ +---+ +---+             |
  |    | 4 | | 5 | | 6 |             |
  |    +---+ +---+ +---+             |
  |    | 7 | | 8 | | 9 |             |
  |    +---+ +---+ +---+             |
  |    | < | | 0 | |OK |             |
  |    +---+ +---+ +---+             |
  +----------------------------------+
  1. Tapez les 6 chiffres du code
  2. Touchez OK (le bouton devient bleu quand les 6 chiffres sont saisis)
  3. Si le code est bon : message vert Tablette appairee et bascule automatique vers l application
  4. Si erreur : effacez avec la fleche retour, demandez un nouveau code au PC Org
Codes d erreur courants :
  • Code expire : plus de 15 minutes depuis l envoi, demandez-en un nouveau
  • Code deja utilise : une autre tablette a deja consomme ce code, demandez-en un nouveau
  • Trop de tentatives, patiente : trop d essais en moins d une minute, attendez 1 minute

Etape 4 - Accepter les autorisations

La premiere fois, le navigateur vous demande plusieurs permissions. Toutes sont indispensables. Touchez Autoriser pour chacune.

Si vous avez refuse une autorisation par erreur : ouvrez les Reglages de la tablette → Safari (ou Chrome) → Field → Autorisations, puis remettez tout sur Autoriser. Voir aussi la section Depannage.

Etape 5 - Verifier la topbar

En haut de l ecran, 3 pastilles indiquent l etat de la tablette. Toutes doivent etre vertes avant de commencer.

  +-------------------------------------------------+
  | FIELD  T01  GP Moto 2026     [GPS] [Net] [Bat]  |
  +-------------------------------------------------+
                                  vert  vert  vert
                                  = pret a partir

Si une pastille est rouge ou orange, voir la section Lire la topbar.

Etape 6 - Premier statut

En bas de l ecran, juste au-dessus de la barre des actions, un bouton affiche votre statut. Touchez-le et choisissez Disponible pour signaler au PC Org que vous etes prete a recevoir une mission.

Ce que voit le PC Org Votre patrouille apparait en vert sur la carte du PC Org, avec le statut Disponible. Vous pouvez recevoir un dispatch a tout moment.

04Lire la carte

La carte occupe la majeure partie de l ecran. Elle vous montre le terrain de l evenement, votre position, et les interventions en cours.

Votre position

Un cercle bleu pulsant au centre de la carte represente votre position GPS en temps reel. Un cercle plus large et translucide autour indique la precision GPS : plus il est petit, plus la position est fiable.

           .-----.
          /       \
         (    *    )      *  = votre position
          \   .   /       () = cercle de precision
           '--+--'

Si vous ne voyez pas votre point bleu : touchez le bouton Me recentrer (cible) en bas a gauche de la carte. La carte revient sur vous et reste centree tant que vous bougez.

Les fiches d intervention sur la carte

Chaque intervention en cours est representee par un pin colore a sa position. La couleur depend de la categorie :

CouleurCategorieExemples
RougePCO.SecoursBlesse, malaise, prise en charge medicale
VioletPCO.SecuriteBagarre, intrusion, vol
OrangePCO.TechniquePanne electricite, fuite, materiel
VertPCO.FluxBouchon, deviation, stationnement

Touchez un pin pour ouvrir la fiche correspondante.

Changer de fond de carte

Le bouton Fond de carte (icone carte, en haut a droite) cycle entre 3 fonds :

  1. Plan (OSM) : carte routiere classique, lisible meme avec peu de reseau
  2. Satellite ACO : photo aerienne haute resolution du circuit, fournie par l ACO
  3. Satellite Egis : alternative satellite a plus large couverture
La vue Satellite ACO est la plus precise pour reperer une zone du circuit (un parking, un acces secours). Le Plan (OSM) est plus rapide en zone faible reseau.

Activer le carroyage

Le bouton Carroyage (icone grille, en haut a droite) affiche une grille avec des reperes alphanumeriques (A1, B2, C3...). Cette grille est utilisee a la radio pour donner une position rapide sans coordonnees GPS.

  +----+----+----+----+
  | A1 | B1 | C1 | D1 |
  +----+----+----+----+
  | A2 | B2 | C2 | D2 |
  +----+----+----+----+
  | A3 | B3 | C3 | D3 |   "Patrouille en C3"
  +----+----+----+----+   = carre de 100m

En zoomant fortement, une sous-grille de 25m apparait pour preciser au quart de carre (ex : C3-2).

Calques (panneau lateral)

Le bouton Calques (en bas, dans la barre des actions) ouvre un panneau ou vous pouvez activer ou masquer :

  • Carroyage 100m / sous-grille 25m
  • Portes et portails (pins cyan)
  • Categories de POI (points d interet : PC, secouristes, parkings, etc.)

Rechercher un lieu

Le bouton Recherche (loupe, en haut a droite) ouvre une zone de saisie. Tapez les premieres lettres d un POI (ex : parc) et touchez un resultat pour zoomer dessus.

Mesurer une distance

En bas a droite, 3 boutons d outils mesure :

  • Mesurer une distance (icone ligne) : touchez 2 points ou plus, la distance s affiche
  • Mesurer une aire (icone carre) : touchez les sommets d une zone, la surface s affiche
  • Mesurer un rayon (icone cercle) : touchez un centre puis un point exterieur

Le bouton poubelle efface toutes les mesures en cours.

Plein ecran

Le bouton Plein ecran (icone fleches) cache les barres du navigateur pour une carte au format maximum. Re-touchez pour sortir.

05Le statut patrouille

Votre statut dit au PC Org ce que vous etes en train de faire. C est l information la plus importante de la tablette : elle determine si vous pouvez recevoir une mission, si vous etes deja engagee, ou en pause.

Ou changer de statut

Le bouton de statut est en bas de l ecran, juste au-dessus de la barre des actions. Il affiche le statut actuel (ex : Disponible) avec un chevron v. Touchez-le pour ouvrir la liste des statuts.

  +-----------------------------------------+
  |              [carte]                    |
  +-----------------------------------------+
  |  +-----------------------------------+  |
  |  |  Disponible                  v    |  |  <- bouton statut
  |  +-----------------------------------+  |
  +-----------------------------------------+
  | [Calques][Missions][Camera][Msgs][SOS] |
  +-----------------------------------------+

Les 4 statuts principaux

StatutSignificationQuand l utiliser
Disponible En ronde, disponible pour recevoir une mission Mode par defaut. Vous patrouillez et attendez un dispatch.
Intervention Engagement sur le terrain, en deplacement vers une intervention Pose automatiquement quand vous acceptez un dispatch ou creez une fiche.
ASL Arrivee sur les lieux - vous etes sur place Pose automatiquement quand vous arrivez a moins de 10m du lieu de l intervention. Vous pouvez aussi le poser manuellement.
Pause Indisponible temporairement Repas, releve d equipage, fin de vacation. Le PC Org sait qu il ne faut pas vous dispatcher.

Le statut "Fin d intervention"

Un 5e statut, Fin d intervention, n apparait que lorsque vous etes en Intervention ou en ASL. Il sert a cloturer proprement une intervention en demandant la liberation au PC Org.

  1. Touchez le bouton de statut
  2. Choisissez Fin d intervention
  3. Une fenetre vous demande un compte-rendu rapide (optionnel)
  4. Touchez Confirmer fin d intervention
  5. Le PC Org est notifie. Vous repassez ensuite en Disponible ou Pause.

Transitions interdites

On ne passe pas directement d Intervention ou ASL a Disponible. La tablette vous force a passer par Fin d intervention (ou Pause), pour que le PC Org sache que l intervention est cloturee. C est volontaire : ca evite qu une intervention disparaisse des ecrans sans compte-rendu. 
  Disponible <-->  Pause
       |              ^
       |              |
       v              |
  Intervention -> Fin d intervention
       |
       v
      ASL  --------> Fin d intervention

Couleurs des statuts

Le statut a une couleur visible sur la tablette ET sur l ecran du PC Org :

  • Disponible = vert
  • Intervention = orange
  • ASL = bleu
  • Pause = gris
  • Fin d intervention = violet
Ce que voit le PC Org Votre patrouille apparait sur la carte du PC Org avec un point de la couleur de votre statut. La liste des patrouilles est triee par statut : interventions en haut, pauses en bas. Le PC Org dispatche en priorite les patrouilles en Disponible.

06Recevoir une mission

Quand le PC Org vous attribue une intervention, votre tablette vous le signale immediatement, meme si vous etiez en train de regarder la carte ou si l ecran etait eteint.

L alerte plein ecran

  +------------------------------------------+
  |  PCO.Secours                       UA    |
  +------------------------------------------+
  |                                          |
  |  Personne au sol pres de la tribune T2,  |
  |  semble inconsciente.                    |
  |                                          |
  |  Zone : Tribune Honneur                  |
  |  Carroyage : C3                          |
  |  Position : 47.95621 N, 0.20543 E        |
  |                                          |
  |  +-----------+      +----------------+   |
  |  |  IGNORER  |      |    ACCEPTER    |   |
  |  +-----------+      +----------------+   |
  +------------------------------------------+

Une fenetre plein ecran s ouvre avec :

  • La categorie de l intervention (Secours, Securite, Technique, Flux) et sa couleur en bandeau
  • Le niveau d urgence en haut a droite (IMP UR UA EU)
  • La description rapide de la situation
  • La zone, le carroyage et la position GPS du lieu
  • Une photo si quelqu un en a deja transmis une

En meme temps, la tablette :

  • Joue un son d alerte (deux tons montants)
  • Vibre 2 fois, puis une vibration longue
  • Recentre la carte sur le lieu de l intervention

Accepter ou refuser

BoutonEffet
ACCEPTER Vous prenez l intervention. Statut bascule en Intervention. La fiche detaillee s ouvre. Le menu d itineraire vous est propose.
IGNORER La fenetre se ferme. Le PC Org est notifie que vous n avez pas pris l intervention et la dispatchera sur une autre patrouille.
Refusez seulement si vous ne pouvez vraiment pas prendre l intervention (deja sur une autre, eloignement majeur). En cas de doute, acceptez et confirmez en radio avec le PC Org.

Lancer l itineraire

Apres avoir accepte, un menu vous propose de naviguer jusqu au lieu :

  +-----------------------+
  |  Itineraire           |
  +-----------------------+
  |  +-----------------+  |
  |  | Itineraire ACO  |  |  <- recommande
  |  +-----------------+  |
  |  +-----------------+  |
  |  |   Google Maps   |  |
  |  +-----------------+  |
  |  +-----------------+  |
  |  |    Annuler      |  |
  |  +-----------------+  |
  +-----------------------+
  • Itineraire ACO : trace une polyline bleue sur la carte de Field, en utilisant les voies de service du circuit. Recommande car il connait les acces dedies.
  • Google Maps : ouvre l application Google Maps de la tablette en mode navigation. Pratique hors enceinte du circuit, mais ne connait pas les voies de service.
  • Annuler : ferme le menu. La fiche reste ouverte, vous pouvez relancer l itineraire plus tard depuis le bouton Itineraire de la fiche.

Arrivee sur place

Le passage en ASL est automatique. Quand votre tablette detecte que vous etes a moins de 10 metres du lieu de l intervention, votre statut bascule tout seul de Intervention vers ASL. Pas besoin de toucher quoi que ce soit. Le PC Org est notifie.

Si la position GPS n est pas tres precise (interieur de batiment, foret), le passage automatique peut tarder. Vous pouvez le forcer manuellement :

  1. Touchez le bouton de statut
  2. Choisissez ASL - Arrivee sur les lieux
  3. Confirmez

Le bandeau d engagement

Tant que vous etes en intervention, un bandeau colore reste affiche en haut de l ecran avec la categorie et la description de la fiche en cours. Touchez-le pour rouvrir la fiche detaillee.

Ce que voit le PC Org Quand vous acceptez, votre patrouille devient orange (Intervention) sur sa carte. Quand vous arrivez a destination, vous passez en bleu (ASL). Le PC Org sait alors que vous prenez en charge la situation et arrete de chercher d autres ressources.

07Creer une fiche

Vous decouvrez vous-meme un probleme sur le terrain (sans avoir ete dispatchee). Plutot que de tout decrire en radio, vous creez une fiche d intervention directement sur la tablette. Le PC Org la voit aussitot et peut envoyer du renfort si besoin.

Quand creer une fiche

  • Vous tombez sur une situation que vous prenez en charge (chute, panne, degradation)
  • Vous voulez que le PC Org garde une trace ecrite et photo
  • Vous estimez que d autres ressources peuvent etre utiles
Pour une urgence vitale (personne inconsciente, agression en cours), declenchez d abord un SOS (voir section 11). La creation de fiche prend 30 secondes, le SOS est instantane et alerte tout le monde.

Ouvrir le formulaire

  1. Touchez Missions dans la barre des actions (en bas)
  2. Touchez + Nouvelle fiche en haut du panneau

Remplir le formulaire

  +-----------------------------------------+
  |  Nouvelle intervention                  |
  +-----------------------------------------+
  |                                         |
  |  Categorie *                            |
  |  +-------+ +-------+ +-------+ +------+ |
  |  |Secours| |Securit| |Techniq| | Flux | |
  |  +-------+ +-------+ +-------+ +------+ |
  |                                         |
  |  Urgence *                              |
  |  +---+  +---+  +---+  +---+             |
  |  |IMP|  | UR|  | UA|  | EU|             |
  |  +---+  +---+  +---+  +---+             |
  |                                         |
  |  Description *                          |
  |  +-----------------------------------+  |
  |  | Personne au sol pres tribune T2.. |  |
  |  +-----------------------------------+  |
  |                                         |
  |  Position : 47.95621 N, 0.20543 E   OK  |
  |  Carroyage : C3                         |
  |                                         |
  |  +-----------+  +--------------------+  |
  |  |  Annuler  |  |   Creer la fiche   |  |
  |  +-----------+  +--------------------+  |

Choisir la categorie (obligatoire)

CategoriePour quoi
PCO.SecoursBlessure, malaise, prise en charge medicale
PCO.SecuriteBagarre, intrusion, vol, suspicion
PCO.TechniquePanne electrique, fuite, materiel a reparer
PCO.FluxBouchon, deviation, stationnement gene

Choisir l urgence (obligatoire)

NiveauSignificationExemple
IMP Implique - pas de blessure, pas de victime, juste a documenter Vehicule mal gare sans occupant
UR Urgence relative - situation stable, pas de danger immediat Panne electrique sur un stand
UA Urgence absolue - engagement prioritaire necessaire Personne blessee consciente
EU Detresse vitale - mobilisation de toutes les ressources Personne inconsciente, arret cardiaque
En cas de doute, choisissez le niveau au-dessus. Il vaut mieux mobiliser un peu trop que pas assez. Le niveau peut etre re-evalue apres par le PC Org.

Description (obligatoire)

Une phrase courte et claire. Ce que voit le PC Org doit suffire a comprendre la situation en 3 secondes.

  • OK : Personne au sol pres tribune T2, semble inconsciente
  • OK : Vehicule en panne sur la voie d acces secours, bloque le passage
  • NON : RAS ou Probleme (pas assez d info pour dispatcher)

Position automatique

La position GPS de la tablette est utilisee comme position de la fiche. Si le carroyage est active, le code (ex : C3) est ajoute automatiquement. Vous n avez rien a saisir.

Si la pastille GPS est rouge au moment de la creation, la position ne sera pas envoyee. Indiquez alors le lieu dans la description (ex : tribune T2 cote nord).

Validation

  1. Touchez Creer la fiche
  2. Toast vert : Fiche creee - Intervention demarree
  3. Votre statut bascule automatiquement en Intervention
  4. Le bandeau d engagement apparait en haut de l ecran
Ce que voit le PC Org Une nouvelle fiche apparait dans sa liste, avec votre nom de patrouille comme creatrice. Sur sa carte, un pin de la couleur de la categorie marque le lieu. Le PC Org peut decider d ajouter du renfort, ou simplement de suivre votre prise en charge.

08Travailler sur une fiche

Une fois engagee sur une fiche (acceptee ou creee par vous), vous pouvez l enrichir au fil de votre intervention : commentaires, photos, codes scannes, et la cloturer a la fin.

Ouvrir la fiche en cours

Plusieurs chemins possibles :

  • Toucher le bandeau d engagement en haut de l ecran (visible tant que vous etes en intervention)
  • Toucher le pin de la fiche sur la carte
  • Ouvrir le panneau Missions et toucher la fiche dans la liste

Ajouter un commentaire

  1. Touchez le bouton Commentaire
  2. Tapez le texte (court, factuel : Victime consciente, douleur jambe gauche)
  3. Touchez Envoyer

Toast vert : Commentaire envoye. Le commentaire apparait dans la chronologie de la fiche, cote PC Org instantanement.

Ajouter une photo

  1. Touchez le bouton Photo de la fiche (ou le bouton Camera de la barre du bas)
  2. Cadrez et touchez le declencheur
  3. Vous pouvez prendre jusqu a 5 photos d affilee
  4. Touchez Joindre a cette fiche (case cochee par defaut puisque vous etes engagee)
  5. Optionnel : ajoutez un commentaire qui accompagne la photo
  6. Touchez Envoyer

Toast vert : Photo envoyee. Photo visible dans la chronologie cote PC Org. La photo est compressee automatiquement (max 1600 px) pour passer meme avec un reseau faible.

Ajouter un scan QR ou code-barres

Pratique pour scanner un badge d acces, un code de site, un QR de materiel.

  1. Touchez le bouton Scan de la fiche
  2. Approchez le code de la camera, le scan est automatique des qu il est detecte
  3. Vous pouvez scanner plusieurs codes a la suite (jusqu a 20)
  4. Si le code est illisible, touchez Saisie manuelle et tapez le numero
  5. Touchez Envoyer

Lire la chronologie

La chronologie de la fiche liste tous les evenements dans l ordre :

  • Creation de la fiche (par qui, a quelle heure)
  • Engagements successifs des patrouilles
  • Tous les commentaires (cote tablette ET cote PC Org)
  • Toutes les photos (avec miniatures cliquables)
  • Tous les codes scannes

C est votre carnet d intervention en temps reel, partage avec le PC Org.

Cloturer la fiche

La cloture d une fiche n est pas la meme chose que la fin d intervention. La fiche reste ouverte tant que le PC Org n a pas decide qu elle etait traitee. Vous pouvez avoir fini votre intervention et passer en Disponible alors que la fiche reste ouverte (par exemple parce que les pompiers prennent le relais).

Fin d intervention (cote tablette)

  1. Sur la fiche, touchez Fin d intervention
  2. Une fenetre vous demande un compte-rendu (optionnel mais recommande)
  3. Touchez Confirmer fin d intervention
  4. Votre statut bascule en Fin d intervention, puis vous pouvez repasser en Disponible

Le compte-rendu est ajoute en chronologie. Le PC Org sait que vous vous liberez et peut redispatcher en consequence.

Ce que voit le PC Org Chaque action que vous faites sur la fiche apparait dans sa chronologie cote PC. C est lui qui decide quand cloturer definitivement la fiche (statut TERMINE), apres avoir lu votre compte-rendu et eventuellement croise avec ceux des autres intervenants.

12Streaming live

Le PC Org peut vous demander de diffuser ce que voit la camera arriere de votre tablette en direct, pendant quelques minutes. Utile pour qu il evalue la situation avec ses propres yeux : ampleur d un attroupement, etat d un blesse, degats sur un materiel.

Comment ca demarre

La demande arrive en pop-up plein ecran avec une vibration et un son d alerte :

  +-------------------------------------+
  |  [video] Flux video demande         |
  +-------------------------------------+
  |  Le PC org demande a voir votre     |
  |  camera en direct.                  |
  |                                     |
  |  - Demande par : Marc (PC org)      |
  |  - Duree maximale : 5 minutes       |
  |  - Sans audio (video seule)         |
  |  - Aucun enregistrement serveur     |
  |                                     |
  |  +---------+  +-------------------+ |
  |  | Refuser |  | Accepter et       | |
  |  |         |  | diffuser          | |
  |  +---------+  +-------------------+ |
  +-------------------------------------+

Ce qui est garanti

  • Aucun audio n est diffuse, jamais. Seule la video.
  • Aucun enregistrement n est fait sur le serveur. Le flux est seulement vu en direct par les operateurs PC Org autorises.
  • La duree est limitee a 5 minutes, apres quoi le flux s arrete tout seul.
  • Vous pouvez arreter a tout moment avec le bouton d arret.

Accepter

  1. Touchez Accepter et diffuser
  2. La camera arriere s active automatiquement
  3. Une bandeau rouge clignotant apparait en haut de l ecran : Flux en cours - X min restantes
  4. Cadrez ce que le PC Org doit voir, comme une video
  5. La carte reste accessible : touchez Carte pour basculer

Refuser

Touchez Refuser. Le flux n est pas demarre, le PC Org est notifie. Toast : Flux refuse.

Refusez si :

  • Vous etes en intervention sur un acte intime (soin, deshabillage)
  • La situation est sensible et merite d etre discutee en radio d abord
  • La tablette est presque dechargee (le streaming consomme beaucoup)

Arreter le flux avant la fin

  1. Touchez le bandeau rouge Flux en cours en haut de l ecran
  2. Touchez Arreter le flux
  3. Le bandeau disparait, la camera s eteint

Le PC Org peut aussi couper le flux de son cote (par exemple s il a vu ce qu il voulait voir).

Limites techniques

Le streaming consomme beaucoup de batterie et de reseau. Si vous etes en zone faible 4G, l image peut etre saccadee ou se figer. Si la batterie passe sous 15%, le flux s arrete automatiquement pour preserver la tablette.

La tablette peut diffuser une seule patrouille a la fois. Si une autre patrouille est deja en train de diffuser, le PC Org doit attendre qu elle ait fini ou couper son flux.

Ce que voit le PC Org Une vignette video en direct apparait dans son tableau de bord, avec votre nom de patrouille, la qualite du signal et le decompte du temps restant. Le flux peut etre re-diffuse vers le mur d ecrans de la salle de gestion, mais jamais enregistre.

16Depannage rapide

Voici les pannes les plus frequentes et leur solution. Si rien ne marche, contactez le PC Org en radio en precisant quel probleme et ce que vous avez deja essaye.

Le GPS est rouge en permanence

  1. Sortez du batiment a decouvert pendant 30 secondes
  2. Verifiez que la localisation est activee dans les reglages systeme de la tablette
  3. Verifiez que Field a la permission GPS (Reglages → Safari/Chrome → Field → Position : Toujours)
  4. Si toujours rouge : redemarrez la tablette (bouton power maintenu 5 sec)

La batterie chute trop vite

  • Baissez la luminosite de l ecran (plus elle est haute, plus ca consomme)
  • Coupez le streaming live si vous l aviez accepte
  • Si vous etes en Intervention ou ASL, le GPS tourne a haute frequence : c est normal de consommer plus
  • Branchez la tablette sur le powerbank de la patrouille

L application rame ou affiche une page blanche

  1. Tirez la page vers le bas pour la rafraichir (geste pull-to-refresh)
  2. Si rien : fermez et rouvrez l application Field
  3. Si toujours rien : redemarrez la tablette
  4. En dernier recours, supprimez le cache du navigateur (Reglages → Safari/Chrome → Effacer historique et donnees) ATTENTION : cela vous deconnecte, il faut re-appairer avec un nouveau code

17Fin de vacation

Quand votre poste se termine (releve par un collegue, fin de service), prenez 2 minutes pour proprement passer la main. Une mauvaise fin de vacation = un dispatch envoye a une patrouille qui n est plus la.

Cas 1 : Releve par un collegue (la tablette continue)

  1. Si vous etiez en Intervention ou ASL, terminez l intervention en cours (voir section Travailler sur une fiche)
  2. Passez le statut en Pause
  3. Notez sur le cahier de releve : votre nom, l heure de fin, l etat de la batterie, la presence ou non de fiches en cours
  4. Briefez le collegue oralement : derniere consigne du PC Org, points chauds du secteur, materiel a surveiller
  5. Le collegue signe le cahier et passe la tablette en Disponible quand il est pret
Pas besoin de se deconnecter. Comme la tablette est appairee a la patrouille (et non a la personne), votre collegue prend la suite sans rien re-saisir.

Cas 2 : Fin du dispositif (la tablette retourne au rack)

  1. Terminez toute intervention en cours
  2. Passez le statut en Pause
  3. Verifiez qu il ne reste rien dans la file d attente de la topbar (compteur orange a zero)
  4. Notez sur le cahier de releve la fin de service et le rendu de la tablette
  5. Branchez la tablette sur le rack de charge prevu au PC

Quand se deconnecter explicitement

Vous pouvez vous deconnecter (effacer le cookie) si :

  • La tablette quitte definitivement le service de l evenement
  • Vous craignez qu elle soit perdue ou volee (le PC Org devra de toute facon la revoquer cote serveur)

Comment se deconnecter

  1. Touchez l icone ? dans la topbar pour ouvrir ce manuel (vous y etes deja !)
  2. Tapez l URL cockpit.lemans.org/field/logout dans la barre d adresse du navigateur
  3. Vous etes redirigee vers la page d appairage
La deconnexion n invalide pas la tablette cote PC Org : le code reste valide jusqu a sa revocation par l admin. Pour vraiment desactiver, demandez au PC Org de revoquer la tablette dans son interface.

Check-list de fin de vacation

  [ ] Statut en Pause
  [ ] File d attente vide (compteur a zero)
  [ ] Fiches en cours cloturees ou transmises
  [ ] Cahier de releve a jour
  [ ] Briefing oral fait au collegue (si releve)
  [ ] Tablette branchee sur le rack (si fin)
